# AV-YKChorus

Juno-60 style ensemble chorus for Akashic Vault, based on TAL-NoiseMaker's
YKChorus by Patrick Kunz. Two chorus voices, independent LFO rates, mix control.

## Structure

```
core/ykchorus/   Vendored YKChorus DSP (GPL-2.0, header-only, unchanged)
src/             IAudioFX wrapper (GPL-3.0)
menu.json        Parameter & menu definition
```
